This folder contains relevant code for the enhanced experiments regarding immunosuppressed patients. This mainly relates to the work of Yu, Fang & Ding (https://link.springer.com/article/10.1186/s40001-025-02622-3). Although their study includes an accessible repository (https://github.com/leanqon/immunocompromised-mortality-ml), the entry selection process with which they build their specific data set is not present.
